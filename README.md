# Amazon Clone
## Overview 
It is a web app made in <b>React js</b>. It basically looks like <b>Amazon</b> with the features of 
* Payment 
* Order
* Login authentication
* Cart
<br/>
Payment processing is done with stripe.

## Prerequisites
* React js
* CSS
* Firebase
